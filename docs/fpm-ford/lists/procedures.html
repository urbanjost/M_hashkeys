Copy 16 32bit words of data from str(pos0) to inp(1:16). The data is padded a requiered by the SHA-256 algorithm. @param str : (in) The message to take a chunk from. @param length : (in) The length of the message in 8bit words. @param inp : (inout) The work area to copy the data to. @param pos0 : (inout) Variable to store the start of the next chunk. @param break : (inout) Indicates the position in the work flow. break=0 on entry -> continue to consume a chunk, pad if needed. break=2 on entry -> continue to consume, padding was allready done. break=1 one exit -> the last chunk was consumed. @param swap : (in) Flag to indicate if swapping to big-endian input (swap=1) should be used. swap=1 is needed for the routine to pass the standard tests, but decreases speed with a factor 2.
